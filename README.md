# Project description
This is a repository to compare the conservation score of essential and non-essential genes across five yeast species

## How to run
```
python script.py
```

## Packages
* json v2.0.9
* csv v1.0
* pandas v1.1.3
* seaborn v0.11.0
* matplotlib v3.3.2
